# QA Automation Assessment

## Tasks Completed:
1. **Mobile Automation:** General Store App using Appium, Mocha, and TypeScript.
2. **API Automation (Bonus):** OrangeHRM Candidate management using Playwright.

## How to Run:
- **Mobile:** Run `npx tsc` then `npx mocha test.spec.js` (Ensure Appium server and Emulator are running).
- **API:** Run `npx playwright test --project=chromium`.
